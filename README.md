## 📌 项目概述 (Project Overview)
本项目是我的个人学术**“数字花园”**与**知识管理系统 (PKM)**。

不同于碎片化的笔记记录，本项目旨在通过严谨的分类学（Taxonomy）构建一个全景式的知识拓扑结构。基于**“理、工、社、文”**四大基石，将加州大学戴维斯分校 (UC Davis) 在读期间的跨学科学习过程，转化为高度结构化、可长期迭代的系统性知识储备。

---

## 🏗️ 知识树架构 (Knowledge Tree Taxonomy)
项目采用顶层 **4x4 正交矩阵分类法**，通过“理论基础 vs 实践应用”与“自然界 vs 人类社会”两个维度进行正交划分：

### 1. 🍃 自然科学 (NATURE / 理科) —— 理论基础
> *探索自然世界的运行规律与底层逻辑。*

* **📐 数学 (MATH):** 核心知识基石。
    * **数学分析进阶:** 从初等微积分、多元微积分延伸至**实分析 (Real Analysis)**、复分析及泛函分析。
    * **代数体系:** 线性代数、近世代数及图论推导。
* **🌌 物理 (PHYSICS):** 包含经典力学、电磁学等核心物理模型。
* **🧬 生命 (LIFE) & 🧪 化学 (CHEM):** 基础生命科学与物质组成原理。

### ⚙️ 2. 工程技术 (TECH / 工科) —— 实践与应用
> *将科学原理转化为改变世界的工具。*

* **💻 计算机 (CS):** 核心技术栈。
    * **底层与架构:** 深入 C/C++ 内存管理、计算机组成原理及高性能计算 (HPC)。
    * **算法设计:** 复杂算法分析、数据结构优化及工程化实现。
* **⛓️ 工程 (ENGINEERING) / 🏥 医学 (MEDICINE) / 🌾 农学 (AGRICULTURE)**

### 📊 3. 社会科学 (SOCIETY / 社科) —— 人类社会规律
> *研究人类行为、组织形态与资源配置。*

* **📈 经济 (ECON):** 重点关注宏微观经济模型与**数理经济分析**。
* **⚖️ 政法 (POLITICS) / 🧑‍🤝‍🧑 社会学 (SOCIOLOGY) / 🧠 心理 (PSYC)**

### 🏛️ 4. 人文学科 (HUMANITIES / 文科) —— 思想与表达
> *关乎人类意义、历史演演进与价值审美。*

* **📖 文学 (LITERATURE) / 📜 历史 (HISTORY) / 🎨 艺术 (ART) / 🤔 哲学 (PHILOSOPHY)**

## 🚀 核心特性 (Key Features)

* **LaTeX 深度集成:** 所有的数学推导与物理模型均采用标准 LaTeX 渲染，确保学术严谨性。
* **跨学科关联:** 利用 Notion 的 Relation 数据库，实现了诸如“图论(数学)”与“计算机网络(工科)”、以及“数理模型(理科)”与“经济分析(社科)”的横向知识连接。
* **结构化笔记逻辑:** 遵循自顶向下的分层架构，确保每一个碎片知识都能在知识树中找到对应的座标。
